* 啟動伺服器
  ```
  $ hexo server
  ```
  或者
  ```
  $ hexo s
  ```
  伺服器預設網址：[http://localhost:4000/](http://localhost:4000/)
<!-- more -->

* 清除產生的頁面
  ```
  $ hexo clean
  ```
* 產生靜態網頁
  ```
  $ hexo generate
  ```
  或者
  ```
  $ hexo g
  ```
* 部署到伺服器
  ```
  $ hexo deploy
  ```
  或者
  ```
  $ hexo d
  ```
